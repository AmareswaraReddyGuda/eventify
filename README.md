# Eventify (Minimal Stack)

**Eventify** is a minimal event-management portal built for college campuses.  
It provides event discovery, one-click student registration with downloadable QR tickets, QR-based check-in for organizers, and a simple organizer dashboard showing registration and attendance metrics.

---

## Key features (MVP)
- User signup / login (Student, Organizer)  
- Public event listing and event detail pages  
- One-click registration with capacity check  
- Unique QR ticket generation (PNG) upon registration  
- Organizer QR scanner page to mark attendance (`html5-qrcode`)  
- Organizer dashboard showing registration counts and attendance %  
- Confirmation email on registration (proof-of-concept); 24-hour reminders are a stretch goal

---

## Tech stack (minimal, team-friendly)
- **Frontend**: HTML5, CSS3 (Flexbox/Grid), JavaScript (Vanilla)
- **Backend**: Flask (Python)
- **Database**: MySQL (via SQLAlchemy + `pymysql`)
- **QR Generation**: Python `qrcode` library
- **QR Scanning**: `html5-qrcode` (client-side JS)
- **Email**: Flask-Mail or SendGrid/Resend API
- **Dev Tooling**: Docker Compose for local MySQL
- **Version Control**: Git + GitHub

## Quick start (development)
> Requirements: Python 3.10+ installed. Docker recommended but optional.

1. **Clone the repo**
```bash
git clone https://github.com/<your-username>/eventify.git
cd eventify
```

2. **Create & activate a Python virtual environment**
- macOS / Linux:
```bash
python3 -m venv .venv
source .venv/bin/activate
```
- Windows (PowerShell):
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

3. **Install dependencies**
```bash
pip install -r requirements.txt
```

4. **Start local MySQL (recommended using Docker Compose)**
- Edit `docker-compose.yml` if you want to change DB password.
```bash
docker compose up -d
```
If you cannot use Docker, install MySQL locally and create a database/user manually.

5. **Configure environment variables**
Copy `.env.example` to `.env` and edit values (do NOT commit `.env` to Git).
Example `.env` contents:
```
FLASK_APP=app.py
FLASK_ENV=development
SECRET_KEY=replace-with-random-string
DATABASE_URL=mysql+pymysql://eventify_user:yourpassword@127.0.0.1:3306/eventify_db
```
Replace `yourpassword` with the password used in `docker-compose.yml`.

6. **Initialize database migrations and create tables**
```bash
# macOS / Linux
export FLASK_APP=app.py

# Windows PowerShell
# $env:FLASK_APP="app.py"

flask db init
flask db migrate -m "initial"
flask db upgrade
```

7. **Run the app**
```bash
flask run
```
Open `http://localhost:5000` in your browser.
